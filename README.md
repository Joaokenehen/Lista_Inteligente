<h1 align="center">🛒 CartFlow - Lista Inteligente</h1>

<p align="center">
  Um aplicativo moderno e intuitivo para transformar a sua experiência de compras no supermercado, calculando gastos em tempo real e mantendo o histórico de tudo o que você consome.
</p>

## 🚀 O Problema que Resolvemos
Chega de se perder no bloco de notas do celular, esquecer itens ou ter surpresas no caixa do supermercado. O **CartFlow** foi desenvolvido para dar fluidez às suas compras, separando o que você ainda precisa pegar do que já está no carrinho, tudo com salvamento automático e suporte a múltiplos usuários.

## ✨ Funcionalidades

- **Gestão de Perfis:** Crie, edite e alterne entre perfis de usuários. Ideal para casais ou famílias que dividem as compras.
- **Lista Inteligente (Auto-save):** Adicione itens em casa. O app salva tudo automaticamente em tempo real. Fechou o app? Nada é perdido.
- **Fluxo de Compras:** Separação clara entre os itens que `Faltam Pegar` e os que já estão `No Carrinho`.
- **Cálculo em Tempo Real:** Insira o valor do item ao colocar no carrinho e acompanhe o valor total da compra atualizando instantaneamente.
- **Histórico de Gastos:** Finalizou a compra? Ela vai para um histórico detalhado com a data e o valor total, permitindo acompanhar seus gastos mensais. Conta com modo de exclusão em massa para gerenciamento.
- **Modo Offline:** Todos os dados são salvos localmente no dispositivo com segurança e rapidez.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as ferramentas mais modernas do ecossistema mobile:

- **[React Native](https://reactnative.dev/)** - Framework principal
- **[Expo](https://expo.dev/)** - Plataforma e ambiente de desenvolvimento
- **[NativeWind (TailwindCSS)](https://www.nativewind.dev/)** - Estilização das interfaces
- **[React Navigation v7](https://reactnavigation.org/)** - Roteamento e navegação
- **[Async Storage](https://react-native-async-storage.github.io/async-storage/)** - Persistência de dados locais
- **[Lucide Icons](https://lucide.dev/)** - Iconografia da interface
- **EAS Build** - CI/CD para geração de binários (APK)

## 📱 Como executar o projeto localmente

Pré-requisitos: Node.js instalado e o aplicativo **Expo Go** no seu celular.

```bash
# Clone este repositório
$ git clone [https://github.com/seu-usuario/cartflow.git](https://github.com/seu-usuario/cartflow.git)

# Acesse a pasta do projeto
$ cd /frontend

# Instale as dependências
$ npm install

# Execute a aplicação
$ npx expo start
```
